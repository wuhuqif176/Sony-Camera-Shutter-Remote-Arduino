SONY 相机快门线控制器程序 (适用于二代机以前型号)
适用于适用mulit 接口控制快门的SONY相机

## 项目描述

一个使用 Arduino 框架和 PlatformIO 开发的，用于控制早期 SONY 相机（如 NEX-5N, α6000 等二代机以前型号）的快门线程序。通过微控制器模拟半按对焦和全按快门信号。

## 功能特性

*   支持半按对焦 (AF)
*   支持全按快门 (Shutter)
*   可编程间隔拍摄 (延时摄影)
*   支持 BULB 门长曝光控制

## 硬件要求

*   **微控制器**:ESP32
*   **相机**: SONY NEX 系列，α5000，α6000 等使用 Multi/Micro USB 快门接口的早期机型
*   **连接线**: 需要一条改装后的 USB 线或专用快门线。**注意：** 引脚定义可能与标准 USB 不同，请务必查阅相机文档以防损坏设备！

## 硬件连接图/表

| 单片机引脚 | 相机快门线接口 |
|------------|----------------|
| GND        | GND            |
| Digital 2  | 对焦 (Focus)   |
| Digital 3  | 快门 (Shutter) |


## 软件依赖

*   [PlatformIO](https://platformio.org/) (推荐) 或 Arduino IDE
*   Arduino AVR 或 ESP8266/ESP32 框架 (根据你的硬件选择)

## 注意事项和警告

*   **风险自负！** 错误的接线或代码可能导致相机损坏。请在充分了解原理并测试后再连接贵重相机。
*   本程序仅适用于特定型号的 SONY 相机，新型号可能使用不同的协议。
*   确保单片机和相机共地。

## 故障排除

*   **相机无反应**: 检查接线，特别是 GND 是否连接可靠。
